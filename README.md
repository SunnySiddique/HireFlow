## 🚀 AI Job Recommendation System (HireFlow)

HireFlow is a premium AI-powered job portal that helps users get personalized job recommendations based on their resume.

---

## 🧠 AI Job Recommendation Flow

![AI Job Flow](/ai-job-flow.png)

The system works by:

- Uploading user resume
- Extracting skills & experience
- Fetching all available jobs
- Using AI to match and rank jobs
- Showing personalized job recommendations
